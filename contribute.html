The Learning Matrix is an open-source, community-driven approach, which means contributing experts create, edit, and adjust the matrix to stay in-touch with modern thinking. If you want to become a contributor, you can join our Github here: https://github.com/HunterLowellDavis/TheLearningMatrix. The data for every matrix choice can be edited in the “TextFiles” folder, with NO coding required.
